GROOVE — Vinyl Record Store
A responsive e-commerce landing page for a vinyl record shop. Built with vanilla HTML, CSS, and JavaScript.

Fully responsive layout (mobile + desktop)
Animated vinyl record on hero section
Product catalog with "Add to cart" functionality
Cart counter in navigation
Genre browser section
Newsletter subscribe form
Toast notifications
CSS animations (spinning vinyl, hover effects)

Tech Stack

HTML5
CSS3 
Vanilla JavaScript (DOM manipulation, event handling)
Google Fonts (Bebas Neue, DM Sans)
